Mastermind Game

This is a game of code breaking skill, where the computer can break the code!

Instructions:

Chose a 4 digit number where each digit is between 1 and 6 (inclusive), write this number down so it isn't forgotten (this is the number that the computer will attempt to find)

The computer will present a series of guesses to you, and you must give feedback during the guess promt in the following ways:

  • Type 'a' for the correct number in the correct place
  • Type 'z' for the correct number in an incorrect place
  • If no numbers match, don't place anything in the feedback line

The computer should find your code in 5 guesses! Click the button to begin.
